# Panda
All-in-One Discord Bot

**Features:** Prefix `.`, Slash Commands, Moderation, Music, Fun, Leveling, Anti-Nuke, etc.

**Prefix Commands:**
.ping  
.help  
.say <message>  
.kick @user  
.ban @user  
.unban <userID>  
.mute @user  
.unmute @user  
.clear <number>  
.userinfo [@user]  
.serverinfo  
.avatar [@user]  
.joke  
.meme  
.roll  
.8ball <question>  
.play <song>  
.skip  
.stop  
.queue  
.whitelist <role>
